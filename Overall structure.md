# **📊 1. 대시보드 (관리자 홈)**

### **상단 요약 카드 4개**

- 오늘 수업 수
- 미확정 수업 수 🔴
- 대강 요청 진행중 🟡
- 이번달 예상 지급액

---

### **중앙: 오늘 수업 리스트 (카드형)**

카드 구성:

```
[초등 코딩 A반]
10:00~12:00 | 강남구 OO센터
강사: 김민수 (확정) 🟢
[상세보기]
```

상태 컬러 규칙:

- 🟢 확정
- 🟡 요청중
- 🔴 미배정
- ⚫ 취소

---

### **하단: 알림 피드**

- 박민수 강사 수락
- 대강 요청 수락됨
- 계약서 서명 완료

---

# **📅 2. 수업 메뉴**

## **2-1. 수업 리스트 화면**

상단 필터:

- 날짜 선택
- 상태 필터(전체/미배정/요청중/확정)

우측 상단 버튼:

- **[+ 수업 만들기]**

리스트는 테이블 or 카드 (MVP는 카드 추천)

---

## **2-2. 수업 생성 화면**

입력 필드:

- 수업명 (텍스트)
- 날짜 (데이트피커)
- 시작/종료 시간
- 장소 (주소 입력)
- 학생명단 업로드 버튼
- 내부 메모

하단 버튼:

- [수업 생성]
- [이전 수업 복제]

---

## **2-3. 수업 상세 화면 (핵심 화면)**

상단:

수업명 / 날짜 / 시간 / 장소

---

### **① 배정 영역**

상태 표시:

- 현재 상태: 미배정 🔴

버튼:

- [강사 추천 보기]
- [강사 수동 선택]
- [대강 요청]

강사 추천 누르면:

→ 가용 강사 리스트 모달

각 강사 카드:

- 이름
- 이번달 근무시간
- 거리(선택)
- [배정 요청]

---

### **② 도착 체크인 영역**

표시 내용:

- 체크인 상태:
    - 미체크인
    - 정상 🟢
    - 의심 🟡
    - 지각 🔴
- 체크인 시간
- 정확도 (예: 12m)
- 장소와 거리 (예: 48m)

지도 없음. 숫자만 표시.

---

### **③ 변경/취소 버튼**

- [강사 변경]
- [수업 취소]

---

# **👩‍🏫 3. 강사 메뉴 (관리자)**

## **3-1. 강사 리스트**

컬럼:

- 이름
- 이번달 근무시간
- 예상 지급액
- 계약 상태(미계약/계약중/만료)

버튼:

- [+ 강사 추가]
- [엑셀 업로드]

---

## **3-2. 강사 상세**

상단:

이름 / 연락처 / 기본 시급

탭 구조:

- 기본 정보
- 가용시간
- 수업 이력
- 계약

---

# **📄 4. 계약 메뉴 (핵심 차별화)**

## **4-1. 계약 리스트**

상태 컬러:

- 초안 ⚪
- 발송 🟡
- 열람 👁
- 서명완료 🟢

버튼:

- [+ 계약 생성]

---

## **4-2. 계약 생성 화면**

상단:

템플릿 선택 드롭다운

입력 필드:

- 강사 선택
- 계약 시작일/종료일
- 시급/회당 금액
- 지급일
- 특약사항(텍스트)

버튼:

- [PDF 미리보기]
- [계약 발송]

---

## **4-3. 계약 상세**

PDF 뷰어 표시

하단 상태:

- 발송일
- 열람일
- 서명일

버튼:

- [PDF 다운로드]

---

# **💰 5. 정산 메뉴**

| 강사명 | 수업수 | 근무시간 | 기본지급액 | 수동수당 | 최종지급액 |

- [엑셀 다운로드]
